# The Nail 1995 - Private Nail Studio Landing Page

Dự án website Landing Page cao cấp được thiết kế riêng cho **The Nail 1995**.

## 🌟 Tính năng
- **Thiết kế Sang trọng (Premium Dark Theme):** Sử dụng phong cách Glassmorphism và màu sắc tinh tế.
- **Responsive:** Hiển thị hoàn hảo trên điện thoại, máy tính bảng và desktop.
- **Tối ưu SEO:** Đã cấu hình meta tags cho Google và Facebook.
- **Tính năng Đặt lịch:** Form đặt lịch tích hợp đầy đủ thông tin dịch vụ và thời gian.
- **Gallery thực tế:** Trưng bày các tác phẩm móng tay thực tế từ tiệm.

## 📂 Cấu trúc dự án
- `index.html`: Cấu trúc nội dung chính của website.
- `style.css`: Toàn bộ giao diện và hiệu ứng chuyển động.
- `script.js`: Xử lý logic menu di động, hiệu ứng cuộn và form.
- `images/`: Chứa các hình ảnh thực tế của bộ sưu tập nail.

## 🚀 Cách chạy dự án
1. Đảm bảo bạn đã cài đặt Python (hoặc bất kỳ web server nào).
2. Mở terminal tại thư mục này.
3. Chạy lệnh:
   ```bash
   npm start
   ```
4. Truy cập: `http://localhost:8080`
